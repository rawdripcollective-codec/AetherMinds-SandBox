# AetherMinds™ — Edge AI Infrastructure for Mobile Agents

AetherMinds™ is a presentation site and scaffold for an "Edge AI" project focused on Planner-Solver-Critic (PSC) architecture for on-device agents. This repository contains the web landing page and the supporting project files.

## What is included

- `index.html` — Landing page (based on the HTML you provided).
- `assets/css/styles.css` — Extracted styles from the original inline CSS.
- `.github/` — Issue & PR templates and a minimal CI workflow.
- `README.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `LICENSE` — Standard repository documentation.

## How to preview locally

1. Clone the repository.
2. Open `index.html` in your browser, or serve it with a static server:
   - Python 3: `python -m http.server 8000` then open `http://localhost:8000`
   - Or use VSCode Live Server extension.
